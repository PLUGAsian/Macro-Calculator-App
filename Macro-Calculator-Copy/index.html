Hello!

Welcome to my macronutrient calculator!

This app is meant to be used as a general estimation as to how many protein and calories you should eat per day. The calculations are derived using the Mifflin St. Joer Equation which is as follows:

  • Male: 9.99 x weight in kilograms + 6.25 x height in centimeters – 4.92 x age + 5
  • Female: 9.99 x weight in kilograms + 6.25 x height in centimeters – 4.92 x age – 161
